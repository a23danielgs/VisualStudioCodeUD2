A Uniform Resource Locator (URL), colloquially known as an address on the Web,[1] is a reference to a resource that specifies its location on a computer network and a mechanism for retrieving it. A URL is a specific type of Uniform Resource Identifier (URI),

Generalmente, el formato de una URL es protocolo://servidor:puerto/directorio/pagina Por ejemplo, https://www.arimetrics.com/blog/. Esto se dividiría en las siguientes partes: Protocolo: indica el esquema de red que usa para recuperar la información. Consiste en una secuencia de caracteres que comienzan siempre por una letra. A pesar de diferenciar entre mayúsculas y minúsculas, suele usarse las minúsculas siempre. Esta secuencia está seguida por dos puntos. Los ejemplos más populares para clasificar URLs son http(usada para transmitir hipertexto), ftp (transferencia de ficheros), mailto (enviar correos electrónicos), file y data (incluye datos). Dos barras (//): esto es requerido sólo por algunos esquemas. Cuando no hay autoridad, no pueden aparecer. Servidor: esta contiene el nombre de la máquina de destino, dividiéndose en el nombre del servidor y el dominio (.es, .com, .org). Puerto: un número que corresponde al punto específico de entrada a la máquina, normalmente asociado al protocolo. Directorio: funciona como los directorios de los sistemas operativos, sirve para organizar las páginas que hay en el servidor. Página: es la página web que se estaba buscando.
